# Setup 

## Prepare env file

```
$ cp ./app/.env.example ./app/.env
```

## Build container

```
$ sh script/dev/build.sh
```

## Install dependencies

```
$ sh script/prepare.sh
```

## Start container

```
$ sh script/dev/start.sh
```

## Test node app

```
$ sh script/test/start.sh
```
